Portofolio Samsoft

Samsoft merupakan pengembang solusi perangkat lunak dan sistem embedded yang berfokus pada integrasi software, firmware, dan hardware untuk kebutuhan bisnis, otomasi, serta perangkat elektronik.

Keahlian

Embedded System Development

Pengembangan firmware dan sistem embedded menggunakan ESP32, STM32, serta mikrokontroler lainnya untuk aplikasi IoT, perangkat kontrol, dan sistem otomasi.

Industrial Automation

Implementasi sistem otomasi industri menggunakan PLC, SCADA, komunikasi serial, serta integrasi perangkat industri untuk monitoring dan kontrol sistem.

Desktop Software Development

Pengembangan aplikasi desktop menggunakan C#, WinForms, dan komunikasi perangkat melalui UART, USB CDC, SPI, dan protokol perangkat khusus.

Hardware Diagnostics & Repair

Analisis kerusakan hardware elektronik, laptop, dan mainboard termasuk troubleshooting regulator, IC controller, serta komponen power dan display.

System Integration

Integrasi software, database, dan perangkat hardware menjadi sistem yang stabil untuk kebutuhan bisnis seperti sistem kasir, perangkat kontrol, dan alat monitoring.

Web & API Development

Pengembangan backend API, integrasi database online, serta dashboard web untuk mendukung aplikasi desktop dan mobile.

Contoh Proyek

Mobile App

Aplikasi Produktivitas Android

Aplikasi utilitas dengan fokus performa ringan, desain sederhana, dan stabil di berbagai perangkat.

Desktop System

Sistem Kasir Desktop

Aplikasi kasir desktop terintegrasi dengan database dan API untuk kebutuhan operasional bisnis skala UMKM.

Embedded Device

Firmware Embedded Controller

Pengembangan firmware perangkat berbasis mikrokontroler untuk komunikasi perangkat, flashing firmware, dan kontrol hardware.
